A Real-Time RGB-D Reconstruction Framework with High-Resolution Texture.

发表于 2020-05-03 |

Abstract：

  We propose the first volumetric RGB-D reconstruction framework based on texture mapping to generate high-resolution texture in real time. The remarkable effect is mainly based on a real-time optimal view selection method, which selects the optimal view of the global mesh triangle by using the optimization with a new patches dense constraints. By storing the optimal view results into voxels and updating them, we can not only solve the problem of global mesh dynamic changes, but also maintain the continuity and consistency of texture patches. To effectively reduce the redundant information of RGB data stored for texture generation, we propose a hierarchical dynamic keyframe selection method to pick up high-quality keyframes from the RGB-D data stream. A downsampling global optimization and a edge based Poisson fusion are designed to accelerate seams leveling. Compared with the on-line(KinectFuison) and off-line(Intrinsic3D, Zollhöfer method) RGB-D reconstruction frameworks, our can achieve a greater clarity texture effect with a lower memory cost and an acceptable calculation cost. More importantly, it provides a new idea for the application of virtual reality and augmented reality.

Reference:
